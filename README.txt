=============================
 Slick Reporter
=============================

Slick Reporter is a small utility to run a (configured) command and based on
the output it will report results to slick.

-----------------------------
 Usage
-----------------------------

To begin using slick-reporter, just run it and use the --slick parameter to
specify the base url for slick, and the --configure parameter to write the
default configuration to a slick-reporter.conf
